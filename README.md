# ⚡ One Click Feedback

Auto-fill IonCudos feedback forms with just a single click.

---

## 🚀 Installation & Setup Guide

Follow these quick steps to get the extension up and running on your computer.

### Step 1: Clone the Repository
Open your terminal or command prompt and run the following command to clone the project:
```bash
git clone https://github.com/hs-hariprasad/One-Click-IonCudos-Form-Fill.git
```
*(Alternatively, download the repository as a ZIP file and extract it to a folder on your PC).*

### Step 2: Load the Extension into Chrome
1. Open Google Chrome and navigate to `chrome://extensions/` (or click the three dots Menu → **Extensions** → **Manage Extensions**).
2. Enable **Developer mode** by toggling the switch in the top-right corner.
3. Click the **Load unpacked** button in the top-left corner.
4. Select the folder where you cloned/extracted the files (ensure it is the folder containing `manifest.json`).

### Step 3: Pin the Extension
1. Click the **Extensions** puzzle icon 🧩 in the top-right corner of the Chrome toolbar.
2. Locate **One Click Feedback** in the list.
3. Click the **Pin** icon next to it so it stays visible on your toolbar for easy access.

---

## ⚠️ Important Note
> [!IMPORTANT]
> If you already have the **IonCudos** form page open in a tab, you **must reload/refresh the page** after enabling or updating the extension before using it.
